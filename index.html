Description

WhatsApp, Facebook and Slack desktop client for OSX and Windows (didn't try yet), based on their official web app Build with Electron.

Download link

BETA v0.1

  • WIN32 Download

To Use

npm install
npm start

or to buid

# Build on OSX
npm run build
# Build on Window
npm run buildwin

Possibility

  • Change Facebook to WhatsApp to Slack
  • Click on link and open it in your <3 navigator
  • Desktop notification
  • Remember you !
  • (working on it) Drag and Drop file to send

Some Idea

  • Shortcut command to altern between the different messenger
  • Possibility to add many slack account If you have any idea ! Please tell me or fork me :)
